# AgroSphere 🌱

AgroSphere is a full-stack, AI-powered agricultural assistant designed to empower farmers with data-driven insights. Built using the MERN stack and integrated with the Gemini API, this platform helps farmers cultivate high-value crops, detect plant diseases via image recognition, and manage operational costs efficiently. 

## 🚀 Key Features

* **AI Disease Detection:** Upload a photo of a struggling crop, and the AI will identify the issue and generate a complete cure plan, including chemical and biological control methods.
* **Smart Cost Calculator:** A dedicated financial tracking tool designed for tenant farmers to manage and forecast end-to-end farming expenses.
* **Visual Product Guide:** Upload a photo of any agricultural product or chemical to instantly receive a detailed description and usage instructions.
* **Crop Planning:** Strategic guidance for farmers looking to transition to high-value crop cultivation.

## 💻 Tech Stack

* **Frontend:** React.js, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **AI Integration:** Gemini API (Large Language Model & Vision)
